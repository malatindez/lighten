Detailed Description

Config class which is used to store all the configuration data for the engine.

Note
This class is a singleton, so you can't create an instance of it, you can only access it through the instance() method
This class is final, so you can't inherit from it
This class is not copyable nor movable
This class is not thread safe

If you want to add a new config value, you have to add it to the config.ini file or use the methods declared in the ini::Ini class.
