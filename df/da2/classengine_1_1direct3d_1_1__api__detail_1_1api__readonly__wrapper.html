Detailed Description

template<typename T>
class engine::direct3d::_api_detail::api_readonly_wrapper< T >

The readonly_d3d_resource_wrapper class is a wrapper for readonly d3d resources.

Template Parameters
TThe type of the resource

This class is used to wrap readonly d3d resources, so that they can be used in the Api class. These resources are protected from accidental modification, and can only be modified by the Api class.
