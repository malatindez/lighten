Detailed Description

The Api class is the main class for the Direct3D11 API.

This class is used to initialize and deinitialize the Direct3D11 API, and to provide global pointers to the most used Direct3D11 objects. This class is a singleton, and can be accessed using the api() function.

Warning
You need to call the engine::core::Engine::Init() function if you want to initialize this class.
You need to call the engine::core::Engine::Deinit() function if you want to destroy this class.
